# Get Started

## Setup

1. Clone the repository.
2. No build process required. Open `index.html` in your browser.

## Making Changes

1. **Copy** `page-template.html` to create a new page.
2. **Link** the new page in `index.html` nav.
3. **Style** using classes from `styles.css`.

## Rules

- **DO NOT** mix Light and Dark modes on the same page unexpectedly.
- **DO** use the `assets/` folder for all images.
- **DO** check `DESIGN-SYSTEM.md` before adding colors.
