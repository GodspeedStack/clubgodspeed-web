# Visual Reference

## Dark Mode Hero (Store)

- **Class:** `.hero-section-dark`
- **Overlay:** `.hero-overlay-dark` (Linear gradient Black 95% -> Transparent)
- **Text:** White (`text-white`)

## Light Mode Hero (Home)

- **Class:** `.hero-section-light`
- **Overlay:** `.hero-overlay-light` (Linear gradient White 95% -> Transparent)
- **Text:** Black (`text-black`)

## Gradients

Ensure gradients provide sufficient contrast for accessibility.

- **Dark:** `linear-gradient(to right, rgba(0,0,0,0.95), transparent)`
- **Light:** `linear-gradient(to right, rgba(255,255,255,0.95), transparent)`
